## 1️⃣ Problem Section (구체적 문제 인식)

### 헤드라인
```

```

```
┌─────────────────────────────────┐
│ "샌드위치 30개 주문이 들어왔는데  │

창업 현장에서 마주하는 진짜 질문들.

```

## 2️⃣ Solution Detail Section (해결책 상세)

### 헤드라인
```
우리는 레시피만 알려주지 않습니다
```

### 서브헤드라인
```
4일 후, 당신은 실제로 주문받을 수 있는 모든 시스템을 갖게 됩니다
```

### 본문 (체크리스트)
```
Table One에서 만드는 것들:

✅ Day 1. 레시피 완성
   → 시그니처 메뉴 개발 & 원가 계산 시트

✅ Day 2. 대량 주문 작업
   → 20인분 작업 타임라인 & 포장 체크리스트

✅ Day 3. 디자인 & 패키징  
   → 스티커/명함 디자인 파일 (실제 인쇄 가능)

✅ Day 4. 웹사이트 구축
   → 주문 페이지 + 결제 연동 (플랫폼 수수료 0%)

━━━━━━━━━━━━━━━━━━━━━━

모든 과정을 1:1로 진행하기 때문에
4일 연속 집중 완성도 가능하고,
주 1회씩 4주로 나눠서 들을 수도 있습니다.

당신의 일정에 맞춰 조정 가능합니다.
```

## 3️⃣ Curriculum Detail Section (상세 커리큘럼)

### 헤드라인
```
4일 완성 커리큘럼
단체 주문 시스템을 완성하는 4단계
```

### 서브헤드라인
```
각 단계마다 실전 산출물이 나옵니다
```

### 본문 (아코디언 형식)

#### [Day 1] 레시피 클래스 🍽️
**무엇을 팔 것인가?**

**세부 내용:**
- 시그니처 메뉴 1종 개발 (또는 기존 메뉴 고도화)
- 식재료 원가 계산 시트 작성
- 메뉴 라인업 구성 (베리에이션 전략)
- 보관 방법 & 유통기한 설정
- 알레르기 성분 표기

**✅ 산출물**: 완성된 레시피 & 원가 계산 시트  
**📍 수강 방식**: 오프라인 출장 수업 / 온라인 1:1  
**💰 단과반 수강료**: [가격]

---

#### [Day 2] 대량 주문 작업 📦
**20인분 이상 어떻게 처리할 것인가?**

**세부 내용:**
- 대량 조리 타임라인 설계 (역산 계획)
- 식재료 발주 수량 & 타이밍
- 포장 준비물 체크리스트
- 배송 vs 픽업 프로세스 정리
- 실제 20인분 작업 시뮬레이션
- 냉장/냉동 보관 전략

**✅ 산출물**: 대량 주문 작업 매뉴얼  
**📍 수강 방식**: 오프라인 출장 수업 / 온라인 1:1  
**💰 단과반 수강료**: [가격]

---

#### [Day 3] 디자인 & 패키징 🎨
**어떻게 보이게 만들 것인가?**

**세부 내용:**
- 브랜드명 & 컬러 선정
- 스티커 디자인 실습 (미리캔버스)
- 포장 박스/용기 소싱처 공유
- 명함 & 리플렛 디자인
- 실제 인쇄 파일 제작 (AI/PDF)
- 온라인 발주 방법

**✅ 산출물**: 인쇄 가능한 디자인 파일 일체  
**📍 수강 방식**: 온라인 줌 1:1 (전국 가능) ✈️  
**💰 단과반 수강료**: [가격]

---

#### [Day 4] 웹사이트 구축 🌐
**어떻게 주문받을 것인가?**

**세부 내용:**
- Framer 웹사이트 제작 (노코드)
- 메뉴 소개 & 가격표 페이지
- 주문 폼 & 결제 시스템 연동 (토스페이먼츠)
- 카톡 알림톡 자동화 설정
- 모바일 최적화 체크
- 도메인 연결 & 런칭

**✅ 산출물**: 실제 주문받는 웹사이트  
**📍 수강 방식**: 온라인 줌 1:1 (전국 가능) ✈️  
**💰 단과반 수강료**: [가격]

## 5️⃣ Process Section (수강 프로세스)

### 헤드라인
```
수강 신청부터 완성까지
```

### 본문 (스텝 플로우)
```
Step 1. 무료 상담 신청
   ↓
   [카톡 상담] 또는 [전화 상담]
   • 목표 확인
   • 일정 조율
   • 출장 여부 확인

Step 2. 수강 신청 & 결제
   ↓
   정규 과정 or 단과반 선택
   • 결제 후 상세 안내 발송

Step 3. 사전 준비
   ↓
   • 줌/출장 일정 확정
   • 사전 준비물 안내
   • 템플릿 & 자료 공유

Step 4. 1:1 수업 진행
   ↓
   • Day별 실습 & 피드백
   • 실시간 문제 해결
   • 산출물 완성

Step 5. 수업 완료 & 사후 지원
   ↓
   • 완성된 시스템 최종 점검
   • 추가 질문 카톡 지원 (2주)
```
