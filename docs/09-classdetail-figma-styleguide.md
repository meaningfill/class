# Table One - Class Detail Figma Style Guide

## Color Styles
- Bg/Default: #FAFAFB
- Surface/Base: #FFFFFF
- Surface/Soft: #F7F8FA
- Border/Default: #ECEFF3
- Text/Primary: #0F172A
- Text/Body: #334155
- Text/Muted: #94A3B8
- Text/Inverse: #F8FAFC
- Accent/Primary: #F472B6
- Accent/Soft: #FDE2F2
- Accent/Border: #F9A8D4
- Navy/Base: #0B1220
- Navy/Surface: #111827
- Navy/Card: #1F2937
- Navy/Border: #2A3342
- Shadow/Soft: rgba(15,23,42,0.18)

## Text Styles (Desktop)
- H1/Fraunces: 48/56, Fraunces Bold
- H2/Fraunces: 32/40, Fraunces Bold
- H3/Fraunces: 24/32, Fraunces SemiBold
- Body/Manrope: 16/26, Manrope Regular
- Body/Small: 14/22, Manrope Regular
- Label/Upper: 12/16, Manrope Bold, letter spacing 0.2em

## Grid & Spacing
- Desktop Frame: 1440
- Grid: 12 columns / margin 80 / gutter 24
- Max container: 1280
- Section padding: 120 (top/bottom)
- Card radius: 24
- Button radius: 999

## Component Naming (Figma)
- Nav/Default, Nav/Mobile
- Breadcrumb/Inline
- Hero/ClassDetail
- Problem/Card, Problem/Grid
- Solution/List, Solution/ImageCard
- Curriculum/Accordion
- Pricing/Card, Pricing/Compare
- Process/Timeline
- CTA/Group
- Footer/Base

## Component Specs (Desktop)
- Nav: 80h / menu gap 32 / CTA 140x40
- Hero: 2-col (6/6), image 520x420
- Problem: 2x2 card grid, card 560x220
- Solution: 7/5 split, list row 88h, image 440x520
- Curriculum: accordion 920w, collapsed 96h
- Pricing: 2-col cards, 620x520
- Process: timeline card 880x120
- CTA: 3 buttons 180x48
- Footer: 4-col grid

## Mobile Overrides
- Frame: 375
- Margin: 20 / gutter 16
- Section padding: 72
- Grid: 1 column
- Card width: 335
- Hero image: 335x240
- CTA buttons: stacked full width
